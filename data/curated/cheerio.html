What is Cheerio?

Cheerio is a fast, flexible, and lean implementation of jQuery designed specifically for the server. It parses HTML and XML documents and provides a familiar jQuery-like API to traverse and manipulate the resulting data structure.

Key Features

  • Familiar Syntax: Uses jQuery-style selectors ($('.class'), $('#id'))
  • Fast: No browser DOM required - pure JavaScript parsing
  • Flexible: Works with HTML and XML
  • Lightweight: Minimal dependencies

Installation

npm install cheerio

Basic Usage

import * as cheerio from 'cheerio';

// Load HTML
const $ = cheerio.load('<html><body><h1>Hello World</h1></body></html>');

// Query elements (jQuery-style)
const heading = $('h1').text();
console.log(heading); // "Hello World"

// Modify content
$('h1').text('Modified!');
$('body').append('<p>New paragraph</p>');

// Get modified HTML
console.log($.html());

Use Cases in RAG Applications

  • Web Scraping: Extract content from web pages for ingestion
  • HTML Cleaning: Strip unwanted elements before creating embeddings
  • Content Extraction: Pull specific sections (articles, main content) from pages
  • Link Extraction: Find and follow links for crawling

Example: Extract Article Content

import * as cheerio from 'cheerio';

async function extractArticle(html: string) {
    const $ = cheerio.load(html);
    
    // Remove unwanted elements
    $('script, style, nav, footer, aside').remove();
    
    // Get main content
    const article = $('article').text() || $('main').text() || $('body').text();
    
    // Clean whitespace
    return article.replace(/\s+/g, ' ').trim();
}
