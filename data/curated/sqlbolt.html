What is SQLBolt?

SQLBolt is a free, interactive SQL tutorial that teaches SQL through hands-on exercises directly in your browser. No setup required - just start learning.

Lesson Overview (18 lessons, ~2 hours)

Basic Queries (Lessons 1-4)

1. SELECT queries 101 - Retrieve data from tables
2. Queries with constraints (WHERE) - Filter results
3. Filtering and sorting (ORDER BY) - Sort results
4. Simple SELECT queries - Review exercises

Multi-table Queries (Lessons 5-7)

5. Multi-table queries with JOINs
6. OUTER JOINs - Include non-matching rows
7. A short note on NULLs

Advanced Queries (Lessons 8-12)

8. Queries with expressions - Calculate values
9. Queries with aggregates (COUNT, SUM, AVG)
10. Queries with aggregates (GROUP BY)
11. Order of execution - How SQL processes queries
12. Subqueries - Nested queries

Data Manipulation (Lessons 13-18)

13. INSERT rows - Add new data
14. UPDATE rows - Modify existing data
15. DELETE rows - Remove data
16. CREATE TABLE - Define new tables
17. ALTER TABLE - Modify table structure
18. DROP TABLE - Remove tables

Key SQL for RAG Apps

-- Find all chunks for a document
SELECT * FROM chunks WHERE document_id = 'doc-123';

-- Join chunks with documents
SELECT c.content, d.title 
FROM chunks c
JOIN documents d ON c.document_id = d.id
WHERE d.org_id = 'org-456';

-- Count chunks per document
SELECT document_id, COUNT(*) as chunk_count
FROM chunks
GROUP BY document_id;

-- Get recent chat messages
SELECT * FROM messages
WHERE session_id = 'session-789'
ORDER BY created_at DESC
LIMIT 10;
